# Navigation Menu Hover Animation

Um componente de menu de navegação minimalista, elegante e totalmente responsivo focado em microinterações visuais desenvolvidas exclusivamente com HTML5 e CSS3. O projeto apresenta uma barra flutuante contendo links essenciais que reagem de forma fluida à interação do utilizador ao passar o mouse (*hover*).

## Tecnologias Utilizadas

* HTML5: Estruturação semântica da barra de navegação através de listas e elementos de link (`<a>`).
* CSS3: Estilização completa, definição de cantos suavizados, tipografia limpa e gerenciamento de sombras.
* Animações em CSS: Utilização de propriedades de transição (`transition`) e pseudo-elementos (`::before`/`::after`) para criar efeitos dinâmicos de sublinhado ou preenchimento durante o hover.

## Funcionalidades e Técnicas Aplicadas

* Design Estilo Card Flutuante: O contêiner principal do menu é desenhado em branco puro com cantos levemente arredondados e uma sombra projetada (`box-shadow`) sutil, conferindo profundidade em relação ao fundo.
* Microinterações no Hover: Configuração de efeitos de transição linear que transformam o aspecto visual dos links (Home, About, Services, Contact) de maneira suave e responsiva ao posicionar o cursor do mouse.
* Alto Contraste Visual: Uso estratégico de um card centralizado claro sobre um plano de fundo azul sólido e vibrante, direcionando o foco do usuário totalmente para as opções de navegação.
* Alinhamento Simétrico com Flexbox: Distribuição horizontal proporcional e perfeitamente equilibrada de todos os links de texto dentro do contêiner.
